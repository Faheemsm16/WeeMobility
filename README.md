# Wee Mobility — Animated Landing Page (Next.js + Tailwind + Framer Motion)

## Quickstart
```bash
npm install
npm run dev
```

Then open http://localhost:3000

## Where to put your assets
- Logo: `public/assets/logo-wee.svg`
- LS-1 images (transparent PNGs): `public/assets/ls1/front-<color>.png`, `side-<color>.png`
- Parts thumbnails: `public/assets/ls1/parts/<part>-<color>.png`
- Brochure: `public/assets/brochures/WEE-LS-1-Brochure.pdf`

## Notes
- Two placeholder images are included for LS-1 (front/side in Misty Grey) so animations work out-of-the-box.
- Replace them with your real renders following the filename scheme in the component.
- Contact form is front-end only (no backend).
